Class: Frame_

Frame_

<private> new Frame_(context, name, alias, path)

Class representing one stack frame.
Parameters:
Name Type Description
context string | undefined Context object, empty in case of global functions or if the browser doesn't provide this information.
name string | undefined Function name, empty in case of anonymous functions.
alias string | undefined Alias of the function if available. For example the function name will be 'c' and the alias will be 'b' if the function is defined as a.b = function c() {};.
path string | undefined File path or URL including line number and optionally column number separated by colons.
